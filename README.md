### Installation

1. Open Hyper's preferences with `Cmd + ,` or edit config file at `~/.hyper.js`.

2. Add `hyper-polarbear`to the plugins list, e.g:

```javascript
plugins: ["hyper-polarbear"];
```

3. Hyper will automatically reload or use (`Cmd + Shift + R`).
